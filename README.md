# 0_1 Meshlet Viewer

![alt text](./dx12_play/img/Viewer.gif "Image")

Simple App to view model meshlets.

Create Meshlets with gen_meshlet (linkki)

`view.exe model.mesh`

| Control               | Action        |
| -------------         |:-------------:|
| Left Mouse + Move     | Rotate        |
| Right Mouse + Move    | Translate     |
| Scroll                | Zoom          |
